# Rick and Morty Dashboard

Dashboard React yang menggunakan Rick and Morty API.

## Features
- Search character
- Filter status
- Responsive design
- Loading spinner
- Error handling

## Technologies
- React JS
- Axios
- Vite
- CSS

## API
https://rickandmortyapi.com/

## Deployment
https://rick-and-morty-dashboard.vercel.app/

## Cara Menjalankan

```bash
npm install
npm run dev
